ES3 Array Object

		
		- 개요
			. 기술 방법 : [ 123, "ABC", "가나다" ]
			. 2의 32승 - 1개 까지 작성 가능
			. Element 추가, 삭제 가능. length 유동적.
			. index : 작성 순서대로(왼쪽부터) 0부터 증번
			. 다차원 배열도 가능 : [[], [], ...], [[[]]....]
			. 생성 방법 : 선언과 동시에 값 기술, new Array(), Array()
			. 초기값 : "," 사이에 기술 없으면 undefined 값으로 초기화
			. length : 변경 가능. 열거나 삭제는 불가능.
			. Sort : Unicode 기준으로 오름차순 정렬. 영문 > 한글. value도 변경됨에 주의.
					 배열의 0부터 다음 인덱스를 비교하여 음수이면 자리를 바꾸는 알고리즘.
					 숫자도 유니코드로 비교하므로, 별도의 정렬 로직 구현 필요.
					 function을 파라메터로 줄수 있는 특성을 이용함.
			. reverse : index를 역순으로 정렬함에 주의. 값은 아님.
			
		cf) ES5 Array - like : {키:값, 키:값, 키:값, 키:값....}
		                       순서 보장 없음. length 속성은 제공. 반복문은 for of로 구현.
