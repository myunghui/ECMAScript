Number object
		
		- Number() : 문자열이 숫자라면 사용 가능
		
		- parseInt() 와의 차이
			. String -> Number Type 변환
			. 자바스크립트에서 Number타입은 64바이트 공간안에 "정수 + 소수 이하" 합하여 넣는 개념 -> 타입 구분 없음
			. parseInt는 글로벌 객체. cf) ES6 : Number.parseInt()
			. parseInt			|Number
			  ---------------------------------------
			  글로벌 객체			|
			  정수값만 반환			|소수값도 유지
			  "123ABC" : 123	|"123ABC" : NaN
			  진수 사용 가능		|진수 사용 불가
			  
		- Number의 상수
			. Number.MAX_VALUE, Number.MIN_VALUE
			. Number.NaN
			. Number.POSITIVE_INFINITY, Number.NEGATIVE_INFINITY
			
		- Number.toFixed() : 고정소수점 표기. 문자열로 반환하는 것에 주의.
