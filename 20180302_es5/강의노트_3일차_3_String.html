String object
		
		- String prototype
		. match : 정규표현식 패턴 매치
		. replace : 패턴 사용 가능
		. search : 패턴 사용 가능
		. split : 패턴 사용 가능
		. substring (인덱스 지정) / substr (문자수 지정)
		. length : String 길이. Number 에는 없는 속성(undefined)
		- 정규표현식
		. /문자열/
		. /문자열/ig : ignore case
		. /문자열/.test() : 검색 결과 존재 여부 반환
		. /문자열/.exec() : 검색 결과 반환
