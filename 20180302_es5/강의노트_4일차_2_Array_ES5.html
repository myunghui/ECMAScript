ES5 Array Object

		
		- Array.prototype
			. forEach : 반복문을 함수로 구현. 반복문내 처리할 로직을 파라메터로 지정.
				-> 제2 파라메터 : function(배열 요소, 인덱스, 배열)
					break, continue 사용 못함.
					return 을 만나면 다음 인덱스의 요소로 이동.
					-> 실행중 배열이 변경되었을때
						요소를 추가하면 처리하지 않음
						현재 인덱스보다 큰 인덱스의 값을 변경하면 변경된 값 사용
						현재 인덱스보다 작은 인덱스의 값을 변경하면 처리하지 않음
				-> 제3 파라메터 : this로 사용할 Object.
			. every	: 반복문의 반환값이 false일때까지 콜백 함수 실행. 함수내 return구문이 없으면 true 반환.
			. some	: 반복문의 반환값이 true일때까지 콜백 함수 실행. 함수내 return구문이 없으면 false 반환.
			. filter	: 반복문의 반환값이 true인 Element만 반환
			. map		: 반복문내 편집한 값을 반환
			. reduce	: 반목문의 직전 인덱스의 처리값을 사용할때.
			. reduceRight : reduce와 역방향
			. indexOf	: 데이터 타입까지 체크. (===)
			. lastIndexOf : 오른쪽부터 탐색하는 indexOf
